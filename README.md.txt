# Password Strength Checker

A simple Python script that checks the strength of a password based on:
- Length (minimum 8 characters)
- Use of uppercase and lowercase letters
- Numbers
- Special characters

## Why it matters
Weak passwords are a major cause of data breaches. This tool helps users (especially small businesses and individuals) create stronger passwords and improve their cybersecurity.

## How to use
1. Install Python (3.7 or later).
2. Run the script:
   ```bash
   python password_checker.py
